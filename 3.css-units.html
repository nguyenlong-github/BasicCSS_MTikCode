CSS UNITS

1. Absolute Units

Kích thước cố định, không bị thay đổi dù các yếu tố xung quanh tác động. Ví dụ, thay đổi kích thước trình duyệt thì phần tử dùng px vẫn giữ nguyên.

px ✅

1px thông thường = 1 điểm ảnh (màn hình thường), nhưng trên màn hình độ phân giải cao có thể chiếm nhiều điểm ảnh hơn.

Ví dụ: Box width = 100px, height = 100px

BOX 1
  • pt
  • cm
  • mm
  • inch
  • pc

2. Relative Units

Kích thước thay đổi và phụ thuộc vào phần tử khác hoặc kích thước trình duyệt.

% ✅

Kích thước theo % dựa vào thẻ chứa nó.

Ví dụ: Box width = 50% của thẻ chứa

BOX 2

rem ✅

Dựa vào font-size của <html> (mặc định 16px).

Width 10rem = 160px

BOX 3

em ✅

Dựa vào font-size của thẻ cha gần nhất. Nếu cha không có thì tìm lên cấp trên.

BOX 4
.parent { font-size: 16px; }
.child { font-size: 2em; /* 32px */ }
.grandchild { font-size: 1.5em; /* 48px */ }

vw ✅

1vw = 1% chiều ngang trình duyệt, không phụ thuộc thẻ chứa.

BOX 5

vh ✅

1vh = 1% chiều dọc trình duyệt.

BOX 6
  • vmin
  • vmax
  • ex
  • ch

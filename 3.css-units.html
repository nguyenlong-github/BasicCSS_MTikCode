CSS UNITS

1. Absolute Unit

Đặc tính của nhóm Absolute unit:
Kích thước sẽ cố định, không bị thay đổi
dù cho các yếu tố xung quanh tác động.
Ví dụ trình duyệt thay đổi kích thước.

  • px ✅

    • px: 1px thông thường chiếm 1 điểm ảnh (đối với độ phân giải thập).
    Đối với độ phân giải cao 1px chiếm nhiều điểm ảnh hơn.

    Ví Dụ: box width 100px và height 100px

2. Realative Units

Đặc tính của nhóm Realative unit:
Kích thước không cố định.
Kích thước sẽ cần một nơi để phụ thuộc vào.

  • % ✅

    • Kích thước được tính theo %,
    dựa vào thẻ chứa nó

    Ví Dụ: box2 width 50% và height 100px

    BOX 2

    Trong trường hợp này, box2 đang nằm trong thẻ ul,
    vì vậy nó sẽ bằng 50% kích thước thẻ ul.
    Giả dụ thẻ ul có width : 100 px -> box2 có width : 50px.


  • rem : root em ✅

    • rem sẽ phụ thộc vào kích thước của thẻ <html>
    Mặc định, trình duyệt thường set font-size của <html> là 16px (trừ khi bạn override).

    BOX 3

    Trong trường hợp này thẻ <html> có font-size mặc định là 16px.
    Vì vậy với width box 3 = 160px (10 * 16px)


  • em ✅

    • em sẽ phụ thuộc vào thẻ gần nhất chứa nó và có thuộc tính fontsize.
    Nếu thẻ gần nhất không có thuộc tính fontsize,
    sẽ dịch chuyển lên thẻ cấp trước đó chứa nó tiếp theo để tìm.

    BOX 4

    Trong trường hợp này box 4 có width = 10em đang nằm trong thẻ div
    có fontsize = 30px.
    Vì vậy với width box 4 = 300px (10 * 30px).
    ⚠️ Trong trường hợp thẻ gần nhất không chứa fontsize nó sẽ dịch chuyển
    đến cấp thẻ trước đó chứa nó.
    ⚠️ em nhân theo cấp → nếu con lại đặt theo em thì nó cộng dồn.

    .parent {

    font-size: 16px;

    }
    .child {

    font-size: 2em; /* 32px */

    }
    .grandchild {

    font-size: 1.5em; /* 1.5 × 32px = 48px */

    }

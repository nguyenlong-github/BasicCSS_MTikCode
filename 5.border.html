Border là lớp viền bao quanh element

Cách 1: Dùng từng thuộc tính


				selector {
					border-width: <độ dày>;
					border-style: <kiểu viền>;
					border-color: <màu sắc>;
				}

Ví dụ: Thêm border 10px, nét đứt, màu đỏ cho Box 1.

Box 1

Cách 2: Viết gộp

				selector {
					border: <độ dày> <kiểu viền> <màu sắc>;
				}

Ví dụ: Thêm border 10px, nét liền, màu blueviolet cho Box 2.

Box 2
* Lưu ý: Mặc định solid có border = 2px nếu không chỉ định độ dày

Border là lớp viền, vỏ ôm lấy element

Cách 1:

selector {
  border-width: <Độ dày>;
  border-style: <Kiểu viền>;
  border-color: <Màu sắc>;
}

Ví Dụ :

Thêm 10px cho border Box 1:
border-width : 10px;
border-style: dashed;
border-color: red;

Box 1

Tạo border có độ dày 10px
border có dạng nét đứt
border có màu đỏ.

Cách 2:

selector {
  border: <độ dày> <kiểu viền> <màu sắc>;
}

Ví Dụ:

Thêm 10px cho border Box 1:
border-width : 10px;
border-style: solid;
border-color: red;

* Lưu ý: Mặc định solid có border = 2px

Box 2

Tạo border có độ dày 10px
border có dạng nét liền
border có màu blueviolet.
